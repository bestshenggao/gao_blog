# 如何阅阅读源码

通过分析，我们已经较为深入的理解了 taro 的整个架构思想和部分内部实现。这些思想值得我们在平时的项目中去实践它。其实看源码的目的是什么，比如我分析 taro init 分析到现在，如果你看完，你会发现有很多很酷的思想，可能在你的世界中，写了几个项目都根本想不起来也可以这样用，看源码的目的就是让你去接触世界上优秀的开源项目是如何设计出来的。从而吸收这些思想，为我所用，使我成长
